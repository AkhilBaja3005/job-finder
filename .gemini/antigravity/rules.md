# Antigravity Rules: Job Finder Workspace

These rules configure Google Antigravity (AGY) workspace specifications and permissions to manage job searches, resume tailoring, and interview preparations.

## 1. Allowed Permissions (Auto-approved Actions)

To enable smooth background execution during automated job scraping and autofilling without prompting you constantly, the workspace grants the following permissions.

```yaml
grants:
  # Allows running the local python backend, tests, and scrapers
  - action: command
    target: "./venv/bin/python *"
  
  # Allows git operations for tracking changes in tailored resume outputs
  - action: command
    target: "git *"
  
  # Allows compiling main.py and testing python setups
  - action: command
    target: "./venv/bin/python -m py_compile *"

  # Allow reading and writing workspace file locations
  - action: read_file
    target: "./*"
  - action: write_file
    target: "./*"
```

---

## 2. LaTeX CV and PDF Compilation Rules

When creating or refining tailored resumes:
*   **Compile Tool:** Resumes must be compiled with **lualatex** or exported using Playwright HTML PDF converters.
*   **Verification Gates:** Before presenting the CV to the user:
    1.  Verify the PDF fits on exactly **1 or 2 pages** (no extra trailing blank pages).
    2.  Check that all numeric grades and GPA/CPI figures are successfully retained and right-aligned (`\hfill`).
    3.  Confirm the ATS text layer matches standard formatting rules (no special character double-escaping issues).

---

## 3. Writing Style and Cover Letter Rubric

All cold emails, connection request notes, and cover letters generated by the agents inside this workspace must follow these rules:
*   **No Em-dashes (`--`) or Hyphens as Sentence Breaks:** Restructure sentences using commas or periods.
*   **Zero-fluff constraint:** Eliminate generic clichés like *"passionate about"*, *"great fit"*, or *"leverage my skills"*. Keep explanations grounded in concrete metrics and achievements.
*   **Forward-looking structure:** Focus on what problems you can solve for the target employer, rather than just repeating CV highlights.
*   **First-person active voice:** Always prefer *"I designed"* or *"I led"* over passive structures like *"a system was designed by"*.

---

## 4. Work rights & Knockouts
*   Always evaluate fit indicators and work authorization gates first.
*   If the job posting explicitly requires citizenship or active security clearances, flag a hard stop or warn the candidate immediately rather than drafting resumes.
